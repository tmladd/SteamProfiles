Either copy the tags of the style you want to use and add your text in-between, or copy it and replace the filler text with your own text!

Header Tag: [h1][/h1]
  • Header: [h1]this text will be extremely large and blue, with an underline.[/h1]
  • Bold Text: [b] this text will be bold [/b] Example: this text will be bold
  • Underlined Text: [u] this text will be underlined [/u] Example: this text will be underlined
  • Italic Text: [i] this text will be italicized [/i] Example: this text will be italicized
  • Strikethrough Text: [strike] this text will be crossed out [/strike] Example: this text will be crossed out
  • Spoiler Text: [spoiler] this text will be blacked out until hovered over [/spoiler]
  • No Parse: [noparse]this text will visibly show tags[/noparse] Example: [strike]this text [b]will[/b] visibly show tags[/strike]
  • Hyperlinks/Link To Site: [url=www.yourwebsitehere.com]This will be the text that has a clickable hyperlink[/url] Example: This will be the text that has a clickable hyperlink
  • Make SURE to nest (format) tags correctly. For example, if you want a bold and italic text, do [b][i]bold and italics[/i][/b], rather than [b][i]bold and italics wrong[/b][/i]. If done improperly, the tags will show up in the text and not behave properly or at all.
